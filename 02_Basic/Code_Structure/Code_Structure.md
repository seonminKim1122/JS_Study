# 코드 구조

### Statement(문)
- statement 는 어떤 작업을 수행하는 '문법 구조'와 '명령어' 를 의미
- 서로 다른 statement 는 세미콜론(;)으로 구분
```javascript
alert('Hello');
alert('World');
```

<br></br>
### 세미콜론
- 세미콜론은 statement 가 끝남을 의미하는데 줄바꿈이 있다면 생략 가능

```javascript
alert('Hello')
alert('World')
```

- 줄바꿈이 있을 때 자바스크립트가 세미콜론을 자동으로 삽입해주긴 하지만 그렇지 못하는 상황도 존재하므로 되도록이면 statement 가 끝날 때 삽입해주기


<br></br>
### 주석
- 한 줄 짜리 주석 : //
- 여러 줄 짜리 주석 : /* */
